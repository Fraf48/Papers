### 2. Exact and Heuristic Techniques for the Traveling Salesman Problem (TSP)
**Type:** Technical Report & Comparative Analysis
**Technologies:** Python, C, CPLEX, Matplotlib

A comprehensive study comparing various algorithmic approaches to solving the symmetric Traveling Salesman Problem (TSP). The project involves implementing and benchmarking algorithms across standard TSPLIB instances.
* **Heuristics:** Nearest Neighbor, Extra-Mileage, 2-OPT refinement.
* **Metaheuristics:** GRASP, Variable Neighborhood Search (VNS), Tabu Search, Genetic Algorithms.
* **Exact Methods:** Branch-and-Cut and Benders' Decomposition using **IBM ILOG CPLEX**.
* **Matheuristics:** Hard-Fixing and Local Branching hybrids.

Link to code: [GitHub](https://github.com/umberto-bianchin/OperationsResearch2.git)
